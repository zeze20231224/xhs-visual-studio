# Install Guide

This guide explains how to save XHS Visual Studio to a desktop or phone for faster access.

Live demo:

https://zeze20231224.github.io/xhs-visual-studio/?v=0.3.0

## Desktop

### Chrome

1. Open the live demo URL.
2. Click the browser menu in the top-right corner.
3. Choose **Cast, save, and share** or **More tools**.
4. Choose **Create shortcut**.
5. Enable **Open as window** if available.
6. Click **Create**.

### Microsoft Edge

1. Open the live demo URL.
2. Click the browser menu in the top-right corner.
3. Choose **Apps**.
4. Choose **Install this site as an app**.
5. Confirm the app name and click **Install**.

## Mobile

### Android Chrome

1. Open the live demo URL.
2. Tap the browser menu.
3. Tap **Add to Home screen**.
4. Confirm the name and tap **Add**.

### iPhone Safari

1. Open the live demo URL in Safari.
2. Tap the **Share** button.
3. Tap **Add to Home Screen**.
4. Confirm the name and tap **Add**.
